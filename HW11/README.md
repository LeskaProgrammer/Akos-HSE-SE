### Чек-лист критериев

Ниже отмечено выполнение всех пунктов задания (10/10):

- [✔] **1. Язык С.** Программа написана на языке C.
- [✔] **2. Компиляция и запуск.** Программа успешно компилируется и запускается.
- [✔] **3. Создание регулярного файла.** В начале работы создается файл с именем "a".
- [✔] **4. Создание цепочки связей.** Реализован цикл, создающий ссылки друг на друга (link_N -> link_N-1).
- [✔] **5. Попытка открытия.** На каждой итерации программа пытается открыть файл через системный вызов `open()`.
- [✔] **6. Определение глубины.** Глубина определяется корректно при возникновении ошибки `ELOOP`.
- [✔] **7. Вывод результата.** Значение максимальной глубины выводится в консоль.
- [✔] **8. Отдельная директория.** Все файлы создаются в папке `recursion_test_lab`, чтобы не засорять текущую директорию.
- [✔] **9. Обработка ошибок.** Проверяются возвращаемые значения (`-1`) для всех системных вызовов.
- [✔] **10. Очистка.** После завершения работы удаляются все созданные ссылки, файл и директория.
