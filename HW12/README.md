### Краткое описание стека
В работе использован симулятор **RARS 1.6** (RISC-V). Графический вывод осуществляется через **Bitmap Display** (Memory Mapped I/O, адрес `0x10010000`).
Для работы с файлами применяются системные вызовы `Open` (1024) и `Read` (63).
